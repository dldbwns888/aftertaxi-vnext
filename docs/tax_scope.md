# 세금 엔진 지원 범위 (Tax Scope)

## 지원하는 세금 규칙

| 규칙 | 근거 | 엔진 구현 |
|---|---|---|
| 해외주식 양도소득세 22% | 소득세법 제118조의2 | `capital_gains_rate=0.22` |
| 연간 기본공제 ₩250만 | 소득세법 제118조의3 | `annual_exemption=2_500_000` |
| 이월결손금 5년 | 소득세법 제118조의4 | `loss_carryforward_krw` 연도별 관리 |
| ISA 비과세 한도 ₩200만 | 조세특례제한법 제91조의18 | `isa_exempt_limit=2_000_000` |
| ISA 초과분 분리과세 9.9% | 조세특례제한법 제91조의18 | `isa_excess_rate=0.099` |
| ISA 연간 납입한도 ₩2,000만 | 조세특례제한법 제91조의18 | `annual_cap=20_000_000` (KRW) |
| 종합과세 누진세 8구간 | 소득세법 제55조 + 지방세 10% | `KOREA_PROGRESSIVE_BRACKETS` |
| 배당 원천징수 15% | 미국 원천징수 (조세조약) | `dividend_withholding=0.15` |
| 배당소득 종합과세 기준 ₩2,000만 | 소득세법 제14조 | `compute_dividend_tax()` |
| 건강보험료 (배당소득 기반) | 국민건강보험법 시행령 제41조 | `compute_health_insurance()` |

## 미지원 (의도적 범위 밖)

| 규칙 | 이유 |
|---|---|
| FIFO/HIFO lot method | AVGCOST 전용 결정 (한국 NTS는 이동평균법 허용) |
| 해외주식 양도소득 분할납부 | 단순화 (일시납 가정) |
| 국내주식 양도세 | 해외 ETF 전용 |
| 금융소득 종합과세 전체 | 배당만 반영, 이자소득 미반영 |
| 연금계좌 (IRP, 연금저축) | TAXABLE/ISA 전용 |
| 증여/상속세 | 범위 밖 |
| 해외 FX 손익 과세 | FX-only 모델 한계 |

## 주의사항

- AVGCOST(이동평균법) 기준. 실제 증권사가 FIFO를 적용하면 결과 차이 발생.
- 환율은 매도 시점 환율 적용 (한국 NTS 기준: 양도일 기준 매매기준율).
- 건보료는 배당소득 기반 MVP. 양도소득은 소득월액 산정 대상 아님 (시행령 제41조).
